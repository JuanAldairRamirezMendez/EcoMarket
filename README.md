# EcoMarket - E-commerce de Productos Ecológicos
EcoMarket es una plataforma de e-commerce especializada en la venta de
productos ecológicos y sostenibles.
## 🚀 Características
- Catálogo de productos ecológicos organizados por categorías
- Sistema de carrito de compras y checkout
- Panel de administración para gestión de productos
- Sistema de autenticación y autorización de usuarios
- Reseñas y valoraciones de productos
- Pasarela de pago integrada
- Responsive design para dispositivos móviles

## 🛠 Stack Tecnológico
- **Backend**: Spring Boot, Spring Security, JPA/Hibernate
- **Frontend**: Angular, TypeScript, Bootstrap
- **Base de datos**: PostgreSQL
- **Despliegue**: Azure App Service, Azure Database for PostgreSQL
- **CI/CD**: Azure DevOps Pipelines
## 📋 Prerrequisitos
- Java 17+
- Node.js 16+
- Angular CLI 12+
- Maven 3.6+
- Docker (opcional)
- Cuenta de Azure (para despliegue)
